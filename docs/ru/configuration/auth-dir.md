# Директория аутентификации

Параметр `auth-dir` определяет, где хранятся токены аутентификации. При запуске команды `login` приложение создаст в этой директории JSON-файлы, содержащие токены аутентификации для ваших аккаунтов Google. Для балансировки нагрузки можно использовать несколько аккаунтов.
